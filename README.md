# DreamDevs 2.0 Bootcamp by Moniepoint

This repository contains all the tasks and projects in the DreamDevs 2.0 bootcamp by Moniepoint, focusing on Java programming and GCP.

## Project Structure

Morning drills are done every day. They can be found in each day\'s folder (`day*/`).

The source codes (`src/`) are a reflection of the files arrangements in [Enum](https://enumverse.com), the LMS used in the bootcamp. For example: `weekendSnacks/` is under Day Three on Enumverse, hence, the file tree below:

```
java-foundations/
├── src/
│   ├── day1/
│   ├── day2/
│   ├── day3/
│       └── weekendSnacks/
│   ├── day4/
│   ├── day5/
│   ├── day6/
│   ├── day7/
│       └── weekendSnacks2/     # path will be updated in src/ once Day Seven is updated on Enum.
│   └── dsa/
├── test/
    └── dsa/
└── README.md
```

## Getting Started

This is an IntelliJ IDEA project. To run the code:

0. Clone the repo and cd.
1. Open IntelliJ IDEA
2. File → Open → Select the `java-foundations` directory
3. Wait for the project to load and dependencies to resolve
4. Run any main class or test using the green play button

## Running Tests

Tests are written using JUnit 5. To run all tests:
- Right-click on the `test` directory
- Select "Run All Tests"
